VST2 Installation on Windows XP / 7 / 8 / 10

Simply unzip the downloaded file and copy the A1AUDIO.de
folder into your VSTPlugins folder. There is no problem having
32 bit AND 64 bit versions installed on the same system, f.e.
if you are using more the one DAW / sequencer / host
application on your machine.
